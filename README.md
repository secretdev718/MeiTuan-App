## Setup

1. **Clone the repo**

  ```
  $ git clone https://github.com/huanxsd/MeiTuan.git
  $ cd MeiTuan
  ```

2. **Install dependencies** (npm v3+):

  ```
  $ npm install
  ```


3. **Running on iOS:**

  ```
  $ react-native run-ios
  ```

## Troubleshooting

> Could not connect to development server

In a separate terminal window run:

  ```
  $ react-native start
  ```
